Taster

Excel-driven E2E Test Management Tool

Excel で記載されたテストケースを Datadog Synthetics / Playwright に登録・実行し、結果を一元管理。
AI によるテストケース自動生成も対応。

Demo を試す Architecture

Features

📝

Excel ベースのテスト管理

TestCases / Steps / InputData シートでテストケースを定義。非エンジニアでも編集可能。

🛠

マルチプロバイダー

Datadog Synthetics API と Playwright の2つのプロバイダーに自動振り分け。

🤖

AI テストケース生成

サイトクローリング / リポジトリ解析から AI がテストケースを自動生成。

📈

結果の時系列管理

テスト結果をタイムスタンプ付きディレクトリに JSON 形式で保存・追跡。

Interactive Demo

Taster の3つのコマンドをインタラクティブに体験できます。

taster init

プロジェクトを初期化し、テンプレートから実行環境を作成します。

Terminal
$ npx taster init my-app --url https://staging.my-app.com

taster read-case

Excel テストケースを読み取り、Datadog / Playwright に登録します。

AI Test Generation

2つの手法で AI がテストケースを自動生成します。

Method A: Site Crawling

Available

URL を指定するだけで、Playwright がサイトを自動巡回。ページ構造を収集し、AI がテストケースを生成します。

URL → Crawl → Analyze → AI Generate → Excel
Basic / Form 認証対応
クロール深度制御
除外パスパターン
プレビューモード

Method B: Repo Analysis

Planning

ソースコードやリポジトリ構造を解析し、ルート定義・コンポーネント構造からテストケースを生成します。

Repo → Detect → Analyze → AI Generate → Excel
Next.js / React Router / Express
コンポーネント構造解析
GitHub API 連携
セレクター候補抽出
